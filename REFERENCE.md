# Reference and Verification

This document provides reference information related to the
**Umm Al Qura Calendar**, the official Islamic calendar system
used in Saudi Arabia.

---

## Reference Usage

The Umm Al Qura Calendar is referenced for:

- Verification of Hijri dates
- Educational and academic research
- Software development and testing
- Calendar alignment and documentation

This repository presents information in a neutral,
documentation-style format for reference purposes.

---

## Official Reference Website

For accurate and up-to-date Umm Al Qura Calendar information,
including Hijri month details and date alignment, refer to:

➡️ https://ummalquracalendar.org/

This website provides documented information aligned with
the officially used calendar system in Saudi Arabia.
